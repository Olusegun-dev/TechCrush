
# TechCrush Backend

## Overview
Backend JavaScript codebase for TechCrush application.

## Tech Stack
- **Runtime**: Node.js
- **Language**: JavaScript

## Getting Started

### Prerequisites
- Node.js (v14 or higher)
- npm or yarn

### Installation
```bash
npm install
```

### Running the Server
```bash
npm start
```
